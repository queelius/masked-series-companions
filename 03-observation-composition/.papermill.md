## Category-Theory Language: CUT

The terms "functors" and "category-theoretic perspective" are removed from the plan. The observe_*() functions in maskedcauses are higher-order functions (closures), not functors in the mathematical sense. Using the term loosely in a statistics paper invites reviewer confusion. The conceptual point (observation schemes compose) stands without the label.

If a rigorous category-theoretic formulation is developed later (defined category, morphisms, non-trivial theorem), it can be a separate result. Do not include as speculation.

## C3 Condition Under Censoring Composition

The censoring composition theorem requires the censoring mechanism to be independent of the failure cause K. Three cases for the censoring time tau:

1. **tau deterministic** (fixed censoring time): C3 trivially satisfied
2. **tau random, parameter-independent** (e.g., administrative censoring): C3 satisfied; state explicitly
3. **tau random, potentially parameter-dependent** (e.g., competing event): must be excluded or given separate treatment

The theorem statement must include a precise measurability condition.

## Implementation Note: observe_mixture()

The `observe_mixture()` function in maskedcauses selects a scheme randomly via `sample.int()` but does not record which scheme was applied to each unit. For likelihood computation, either:
- The practitioner knows which observation type each unit received, OR
- The likelihood marginalizes over scheme assignment

Under independence, the mixture likelihood is the weighted average of individual likelihoods. State and prove this. Note whether the current implementation supports it or requires extension.
