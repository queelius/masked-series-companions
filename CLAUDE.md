# CLAUDE.md

## Project Overview

This is a **research planning repository** collecting companion paper ideas for the masked series systems likelihood framework. Each subdirectory contains a research brief for a potential paper.

## Related Repos

- **Foundation paper**: `~/github/papers/masked-causes-in-series-systems/`
- **Exponential companion**: `~/github/papers/expo-masked-fim/`
- **Model selection**: `~/github/rlang/reliability-estimation-in-series-systems-model-selection/`
- **Original thesis**: `~/github/papers/reliability-estimation-in-series-systems/`
- **R packages**: `~/github/rlang/maskedcauses/`, `~/github/rlang/wei.series.md.c1.c2.c3/`

## Conventions

- Each research direction lives in a numbered directory (`01-`, `02-`, ...)
- `brief.md` in each directory contains: motivation, core result sketch, scope, open questions, target venue
- When a direction matures, spin it off into its own repo under `~/github/papers/`
- Reference the foundation paper's notation (theta, h_j, R_j, C1-C2-C3) consistently
